# /init-session コマンド

セッションを初期化し、前回の作業状態を復元する。
SessionStart hookでも自動実行されるが、手動でも呼び出せる。

## 実行手順
1. 本日の日付を確認する
2. 最新セッションファイルを読み込む:
   Glob ツールで `.claude/memory/sessions/*.tmp` を検索し、
   ファイル名（YYYYMMDD）が最大のものを選択して Read ツールで読み込む。
3. `.claude/memory/memory.json` を Read ツールで読み込む（存在する場合）
4. `.claude/instincts/clusters/` の全JSONファイルを Glob ツールで検索し、
   存在する場合は Read ツールで読み込む
5. 以下の形式でサマリを表示する:

---
## セッション再開 ({今日の日付})

### 前回セッション ({前回の日付})
**残タスク:**
{残タスクリスト}

**前回うまくいったこと:**
{成功アプローチの要約}

**注意事項（前回失敗したこと）:**
{失敗アプローチの要約}

### 有効なインスティンクト
{プロジェクト固有インスティンクト数}件のパターンが読み込まれました

---

続きから作業しますか？それとも新しいタスクを開始しますか？
