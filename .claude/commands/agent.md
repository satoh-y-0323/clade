# /agent コマンド

指定したエージェントを起動する。エージェントに応じたルール・スキル・権限が適用される。

## 使い方
```
/agent:architect   # 設計・アーキテクチャ担当
/agent:developer   # 実装・テスト・デバッグ担当
/agent:reviewer    # コードレビュー・品質確認担当
```

## 起動手順（このコマンド実行時に行うこと）
1. `$ARGUMENTS` からエージェント名を取得する（architect / developer / reviewer）
2. 対応するエージェント定義を読み込む: `.claude/agents/{name}.md`
3. 対応する役割別ルールを読み込む: `.claude/rules/{name}/{name}.md`
   - 役割別ルール内の `@` インポートで個別ルールも自動ロードされる
4. 使用可能スキルを確認して適用する
5. 以下の形式でエージェント起動を宣言する:

---
## エージェント起動: {name}

- 役割: {役割の説明}
- 権限: {権限一覧}
- 適用ルール: {読み込んだルールファイル一覧}
- 使用可能スキル: {スキル一覧}
- モデル: {推奨モデル}

準備完了です。何をお手伝いしますか？
---
