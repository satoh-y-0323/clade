# /init-session コマンド

セッションを初期化し、前回の作業状態を復元する。
SessionStart hookでも自動実行されるが、手動でも呼び出せる。

## 実行手順
1. 本日の日付を確認する
2. 最新セッションファイルを読み込む:
   ```bash
   ls -t .claude/memory/sessions/*.tmp 2>/dev/null | head -1
   ```
3. `~/.claude/memory/memory.json` を読み込む（存在する場合）
4. `.claude/instincts/clusters/` の全JSONファイルを読み込む（存在する場合）
5. 以下の形式でサマリを表示する:

---
## セッション再開 ({今日の日付})

### 前回セッション ({前回の日付})
**残タスク:**
{残タスクリスト}

**前回うまくいったこと:**
{成功アプローチの要約}

**注意事項（前回失敗したこと）:**
{失敗アプローチの要約}

### 有効なインスティンクト
{プロジェクト固有インスティンクト数}件のパターンが読み込まれました

---

続きから作業しますか？それとも新しいタスクを開始しますか？
