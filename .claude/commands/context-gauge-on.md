# /context-gauge-on コマンド

コンテキスト使用率ゲージをステータスラインに表示する。
`.claude/settings.json` に `statusLine` 設定を追加する。

## 実行手順

1. Read ツールで `.claude/settings.json` を読み込む
2. 既に `statusLine` キーが存在する場合は「コンテキストゲージはすでに有効です。」と報告して終了する
3. 存在しない場合、Edit ツールで `settings.json` の末尾の `}` の直前に以下を追加する:

```json
  "statusLine": {
    "type": "command",
    "command": "node .claude/hooks/statusline.js"
  }
```

4. 追加後、以下をユーザーに伝える:
   「コンテキストゲージを有効化しました。Claude Code を再起動すると反映されます。」

## ゲージの見方

```
[██████████] 0%    ← 全セル緑、余裕あり
[███████░░░] 70%   ← 右3セルが赤、60%超で黄色表示
[██████░░░░] 60%   ← ちょうど60%
[█████░░░░░] 100%  ← 全セル赤、90%超で赤表示
```

- 全10セル、1セル = 10%
- 使用率が増えるにつれ右から赤くなる
- 数値色: 60%超→黄、75%超→オレンジ、90%超→赤
