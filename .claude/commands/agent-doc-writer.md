# /agent-doc-writer コマンド

ドキュメントライターエージェント（doc-writer）をサブエージェントとして起動する。

## 動作
Agent ツールで `subagent_type: doc-writer` を指定して起動する。

## 用途
- Mermaid 図の生成（フロー図・クラス図・ER図・シーケンス図）
- README の作成・更新
- 操作手順書・運用手順書の作成
- API仕様書の作成
- 既存コードからの仕様書逆引き生成

## 生成されるファイル
出力先はヒアリング時にユーザーが選択する:
- `.claude/reports/doc-*.md`（一時保管）
- プロジェクト内の任意のパス（例: `docs/architecture.md`）
- 表示のみ（ファイル保存なし）

## 注意
- 標準ワークフロー（フェーズ構成）とは独立して動作する
- このコマンドは単独で完結する（他エージェントへの連携は不要）
- ソースコードの変更は行わない
