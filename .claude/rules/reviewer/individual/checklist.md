# Review Checklist

## 必須チェック項目

### セキュリティ
- [ ] SQLインジェクション・XSS等の脆弱性がないか
- [ ] 認証・認可が適切に実装されているか
- [ ] 秘密情報がハードコードされていないか
- [ ] 入力値バリデーションが実装されているか

### 品質
- [ ] エラーハンドリングが実装されているか
- [ ] ログ出力が適切か（個人情報をログに出していないか）
- [ ] テストが追加・更新されているか
- [ ] 既存テストが全て通過するか

### 保守性
- [ ] 関数・変数名が意図を表しているか
- [ ] 重複コードがないか（DRY原則）
- [ ] コメントが必要な複雑なロジックに付いているか

## 推奨チェック項目
- [ ] パフォーマンスへの影響がないか（N+1クエリ等）
- [ ] 依存ライブラリの追加が妥当か
- [ ] ドキュメントが更新されているか
